🎧 About NexTune X

NexTune X is a modern Android music player designed to provide a smooth, beautiful, and powerful audio experience.

Built using modern Android technologies and Material 3 design principles, NexTune X focuses on performance, usability, and a premium listening experience.

⚙️ Build Instructions

Clone the repository:

git clone https://github.com/rahman2008-svg/NexTune-X.git
cd NexTune-X

Build Debug APK:

chmod +x gradlew
./gradlew assembleDebug

Build Release APK:

./gradlew assembleRelease
